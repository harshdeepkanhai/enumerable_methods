# Enumerable methods

> Custom Enumerable methods with following methods

- `my_each`
- `my_each_with_index`
- `my_select`
- `my_all?`
- `my_any?`
- `my_none?`
- `my_count`
- `my_map`
- `my_inject`

## Built With

- Ruby
- Rubocop

## Live Demo

[Live Demo Link](https://repl.it/@kanhai/enumerable)


## Getting Started


To get a local copy up and running follow these simple example steps.

### Prerequisites

- `Ruby version 2.7`
- `Rubocop` gem
- `bundler` gem installed

### Setup

- clone this repo using `git clone`
- run `bundle install`

### Usage

- export the library by `require './lib/enumerables.rb'`


### Run tests

- run `rubocop` in the terminal
- run `rspec spec/enumerables_test.rb`
