# Enumerable methods

> Custome Enumerable methods with following methods

- `my_each`
- `my_each_with_index`
- `my_select`
- `my_all?`
- `my_any?`
- `my_none?`
- `my_count`
- `my_map`
- `my_inject`

## Built With

- Ruby
- Rubocop

## Getting Started


To get a local copy up and running follow these simple example steps.

### Prerequisites

- `Ruby version 2.7`
- `Rubocop` gem

### Setup

- clone this repo using `git clone`

### Usage

- add some test cases and
- run `ruby test.rb`

### Run tests

- run `rubocop` in the terminal
